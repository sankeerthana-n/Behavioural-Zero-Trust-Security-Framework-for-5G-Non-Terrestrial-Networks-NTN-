# Behavioural-Zero-Trust-Security-Framework-for-5G-Non-Terrestrial-Networks-NTN-
This project simulates a Space‑Air‑Ground Integrated Network (SAGIN) with HAPS, LEO, MEO, and GEO satellites. It implements a Behavioural Zero‑Trust model that continuously verifies nodes using telemetry (location, velocity, identity) and prevents spoofing, replay, and impersonation attacks.
